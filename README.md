# Tcp-Chatroom-Using-Python
The TCP Chatroom using Python is a real-time, multi-user communication system developed using Python’s built-in socket and threading libraries. It operates on a client–server architecture, where a central server establishes a TCP connection, listens for incoming client requests, and manages multiple users concurrently. TCP (Transmission Control Protocol) ensures reliable, ordered, and error-free delivery of messages, making the communication stable and consistent. Each client connects to the server using a unique socket, sends a nickname or login credentials, and can exchange messages with other connected users in real time.

The server handles multiple clients simultaneously using multithreading, where a separate thread is created for each connected user. When a client sends a message, the server processes it and either broadcasts it to all connected users or delivers it privately if a specific username is mentioned. The system also includes proper connection handling, detecting when users disconnect and notifying others in the chatroom. Advanced implementations may include features such as user authentication, message history storage, UTF-8 encoding support for special characters, thread-safe data management, and structured user mapping using dictionaries.

This project demonstrates core networking concepts including socket lifecycle management (bind, listen, accept, send, receive, close), concurrency, synchronization using locks, and error handling mechanisms. It provides practical experience in building scalable communication systems and forms a foundation for more advanced applications such as encrypted messaging systems, GUI-based chat platforms, cloud-hosted communication services, and asynchronous high-performance servers using modern networking frameworks.
